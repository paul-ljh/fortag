Entire application was written, run, tested in MacOS JAVA 12.

Type "make" to compile the application
Type "make run" to run the application

Directory breakdown:
"imageHistory.txt" holds paths of previously loaded images

"/assets" contains testings images, icons used in the app.

"/common_components" contains service files, helpers, constants, plus a third-party component "WrapLayout.java" from https://tips4java.wordpress.com/2008/11/06/wrap-layout/

"/image_components" contains components used to display a single image

"/toolbar_components" contains components used to display the toolbar

"/views" and "/models" contains view and model classes
